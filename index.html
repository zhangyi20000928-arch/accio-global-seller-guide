What is Accio Work?

Accio Work is your desktop AI workstation for cross-border business. Imagine a team of AI agents sitting on your computer, ready to handle market research, listings, marketing, customer follow-ups, and store operations -- all in plain language, no coding required.

🔗

How to install Accio Work

Download: accio.com/work
Pick your OS (macOS or Windows), download, install, sign up with your email, verify with the code in your inbox -- that's it. You can change the interface language anytime under Avatar → Language.

✅ Already have Accio Work? Skip ahead to Section 2 to install the Global Seller Agent.

02

Enable the Global Seller Agent Plugin

The Global Seller Agent is the agent on Accio Work made for Alibaba.com sellers. It connects to your store, knows your products and buyers, and uses 20+ pre-loaded professional skills -- so you don't need to configure anything yourself.

💡

A familiar name with a new home. If you've used Smart Assistant on Alibaba.com before, you'll feel right at home -- it's been renamed to Global Seller Agent inside Accio Work, with even more powerful capabilities.

STEP 1

Switch to the official AI agent in the new message and complete the authorization.

On the Accio Work home page, click New Message → select Global Seller Agent. If you haven't authorized yet, an authorization prompt will appear → click "Go to bind" to be redirected to the authorization page → log in with your Alibaba.com account to complete authorization. Once authorized, your store data, product catalog, and inquiry records will be connected — the agent now knows your business.

STEP 2

Start chatting

Head back to chat and start a conversation with Global Seller Agent. Just type what you need in your own words. If you're using Alibaba.com capabilities with a non-official agent, make sure the Global Seller Agent plugin is turned on in the toolbar -- the plugin must be active for the agent to access your Alibaba.com data and skills.

⚠️ The number of active plugins affects guided prompts. When only the Global Seller Agent plugin is enabled, the chat will display Alibaba.com-specific guided prompts (e.g. Bulk product upload, Product optimization). If multiple plugins are enabled at the same time, these prompts will not appear. For the best guided experience, keep only the Global Seller Agent plugin turned on.

💡 Tip: Just click the suggested prompts under the chat box -- the agent will automatically use the right skill for you. No need to memorize anything.

STEP 3

Enable Alibaba.com capabilities in other agents

The Global Seller Agent plugin is not exclusive to the official agent. Other agents in Accio Work -- including custom-built agents -- may also be bound to this plugin to access Alibaba.com capabilities such as store data queries, product analysis, and buyer inquiry handling.

How to bind: Go to Plugins → open the Global Seller Agent detail page → find the Applied to Agents section → click Manage to add or remove agents.

Pro Tips: Make Every Credit Count

Credits Spent = Model Compute Cost + Tokens Used

💡

Use the right model for the job

Daily routine tasks (drafting messages, simple lookups)? Use a lighter model. Complex tasks (full store diagnostics, deep market analysis)? Switch to an advanced model. You can pick the model right in the chat box.

🎯

Keep your prompts focused

Ask one clear thing at a time. Don't pile 5 unrelated questions into one message -- break them up. Bonus: tell it the length you want ("top 3" / "in 100 words") to save tokens.

🧰

Don't over-load on skills

Only enable the skills you actually use. Loading every available skill makes the agent slower and more expensive to run.

⏹️

Stop runaway tasks

If a task is taking unusually long -- a deep analysis, a large batch operation, or a chain that's clearly going off course -- you don't have to wait it out. Click the Stop button in the chat to halt execution immediately.
